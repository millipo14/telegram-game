# Browser RPG (JavaScript)

Интерактивная браузерная RPG-игра, основанная на механике броска кубика (d20), модификаторах характеристик и системе исходов действий.

Проект реализует базовую игровую логику, вдохновлённую RPG-играми в стиле Dungeons & Dragons.


## Технологии

- HTML5
- CSS3
- JavaScript (Vanilla)

## Описание проекта

Проект представляет собой браузерную мини-игру, в которой пользователь:

- выбирает персонажа
- выполняет действия с определённой характеристикой
- бросает виртуальный кубик d20
- получает результат на основе игровой логики


## Игровая механика

- Бросок кубика d20
- Модификаторы характеристик персонажа
- Проверка сложности действия (DC)
- Критические успехи и критические провалы
- Разные исходы в зависимости от результата броска

## Реализованный функционал

- Система классов персонажей
- Характеристики
- Расчёт модификаторов характеристик
- Генерация случайного результата
- Определение исхода действия
- Обновление UI в зависимости от результата
- Передача результата между страницами

## Особенности проекта

- Реализация игровой логики на чистом JavaScript
- Работа с состоянием игрока в браузере
- Разделение игровой логики и отображения интерфейса
- Мини-архитектура с взаимодействием между страницами

## Примечание

Проект создан в учебных целях для практики JavaScript и реализации игровой логики в браузере.
